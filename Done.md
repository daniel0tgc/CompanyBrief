## Architecture Decisions

- Platform: Web (Next.js 14 App Router) — not React Native
- Auth: NextAuth v5 (Auth.js) with Google provider → httpOnly JWT cookie; `NEXTAUTH_SECRET` shared with Fastify for server-to-server token verification
- No Stripe — free product
- AI: Anthropic `claude-sonnet-4-6` with tool use, runs in BullMQ workers only
- SSE bridge: BullMQ job → Redis pub/sub → Fastify SSE route → browser EventSource
- SSE auth: NextAuth token passed as `?token=` query param (EventSource cannot set headers)
- DB: PostgreSQL + JSONB for `companies.analysis` — no MongoDB
- Agent system prompt: loaded from `ANALYSIS_CONTEXT.md` file at startup, cached in module scope
