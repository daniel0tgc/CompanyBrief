## Project Overview

**App:** CompanyBrief
**One-line goal:** Type a company name, get a full AI-generated market research brief with a contextual chatbot
**Target user:** Anyone doing market research — investors, operators, job seekers, students, analysts
**Core value:** Eliminates hours of manual research; existing tools (ChatGPT, Google) require you to know what to ask — CompanyBrief asks the right questions for you
**Monetization:** Free
**Platform:** Web (Next.js 14 App Router)
**Start method:** Scaffold from scratch
**Reason:** Web-native product (sidebar layout, SSE streaming, expandable cards). No mobile. No Stripe. React Native base is not applicable.

## Database Schema

```typescript
// packages/api/src/db/schema.ts

import { pgTable, uuid, text, timestamp, jsonb } from 'drizzle-orm/pg-core'

export const users = pgTable('users', {
  id: uuid('id').primaryKey().defaultRandom(),
  email: text('email').notNull().unique(),
  googleId: text('google_id').unique(),
  displayName: text('display_name'),
  avatarUrl: text('avatar_url'),
  createdAt: timestamp('created_at').defaultNow().notNull(),
  updatedAt: timestamp('updated_at').defaultNow().notNull(),
})

// companies.status: 'pending' | 'running' | 'complete' | 'error'
// companies.analysis: null until complete, then the full JSON object
export const companies = pgTable('companies', {
  id: uuid('id').primaryKey().defaultRandom(),
  userId: uuid('user_id').notNull().references(() => users.id, { onDelete: 'cascade' }),
  name: text('name').notNull(),             // e.g. "Vercel"
  slug: text('slug').notNull(),             // e.g. "vercel" (lowercased, URL-safe)
  status: text('status').notNull().default('pending'), // pending | running | complete | error
  analysis: jsonb('analysis'),              // null until complete — shape defined below
  errorMessage: text('error_message'),      // populated if status = 'error'
  createdAt: timestamp('created_at').defaultNow().notNull(),
  updatedAt: timestamp('updated_at').defaultNow().notNull(),
})

// conversations.messages: array of { role: 'user'|'assistant', content: string, createdAt: string }
export const conversations = pgTable('conversations', {
  id: uuid('id').primaryKey().defaultRandom(),
  companyId: uuid('company_id').notNull().references(() => companies.id, { onDelete: 'cascade' }).unique(),
  messages: jsonb('messages').notNull().default([]),
  createdAt: timestamp('created_at').defaultNow().notNull(),
  updatedAt: timestamp('updated_at').defaultNow().notNull(),
})

// expansion_cards: one row per chatbot follow-up that surfaces new information
// sectionKey maps to the analysis section this card is anchored to (e.g. 'competitive_position')
export const expansionCards = pgTable('expansion_cards', {
  id: uuid('id').primaryKey().defaultRandom(),
  companyId: uuid('company_id').notNull().references(() => companies.id, { onDelete: 'cascade' }),
  sectionKey: text('section_key').notNull(), // e.g. 'competitive_position', 'bull_case', 'general'
  question: text('question').notNull(),       // the user's question that triggered this card
  content: text('content').notNull(),         // the assistant's answer (markdown)
  createdAt: timestamp('created_at').defaultNow().notNull(),
})
```

**Analysis JSONB shape (TypeScript type — define in `packages/api/src/lib/types.ts`):**
```typescript
export type CompanyAnalysis = {
  tagline: string
  what_they_do: string[]
  problem_solved: string[]
  ai_angle: string[]
  competitive_position: string[]
  competitors: { name: string; notes: string }[]
  customers: { category: string; examples: string[] }[]
  market_attractiveness: string[]
  disruption_risks: string[]
  future_outlook: string[]
  bull_case: string
  bear_case: string
  feedback: { pros: string[]; cons: string[]; source_url: string }
  doc_url: string
}
```

**Relations:**
- `companies.userId` → `users.id` (cascade delete)
- `conversations.companyId` → `companies.id` (cascade delete, unique — one conversation per company)
- `expansionCards.companyId` → `companies.id` (cascade delete)
